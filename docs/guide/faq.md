# 常见问题

## 1. URL 预览为什么失败/空白？

组件通过 `axios` 在浏览器发起 `GET` 请求并读取 `blob`，因此 URL 必须满足以下条件：

- 资源可从浏览器访问
- 跨域时服务端已配置 CORS（至少允许 `GET` 与响应头）

若你的资源需要鉴权或无法配置 CORS，推荐使用 [Iframe 嵌入（推荐）](/guide/iframe)。

## 2. 为什么必须给父容器设置高度？

预览器默认使用 `width: 100%`、`height: 100%` 填满父容器；如果父容器高度为 `auto`，渲染区会塌陷。
