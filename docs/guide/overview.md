# 概述

Flyfish Viewer 是一款纯前端（Serverless）的全能文档预览器：文件解析与渲染全部在浏览器端完成，不会给服务器带来额外压力。

## 特性

- 支持海量文件格式：`docx` / `doc` / `pptx` / `xlsx` / `xls` / `csv` / `pdf` / `markdown` / 图片 / `mp4` / 纯文本
- 纯前端方案：无需后端转码服务，可部署在任意静态站点或容器
- Web Worker 异步解析：尽量避免阻塞主线程
- TypeScript 模块化实现：便于二次开发与定制

## 适用场景

- OA/流程系统中的附件预览
- 网盘/知识库/企业文档中心
- 需要在浏览器端离线解析文件的应用
